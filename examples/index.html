With Maximum and Minimum Dates Set

Note the code below uses a Date object only to keep with the present date.

var mind = new Date(), maxd = new Date(); maxd.setDate(maxd.getDate()+7); mind.setDate(mind.getDate()-7); $('#w_date_range').chronos({ minDate: mind, madDate: maxd });

Accessing Methods

Methods on Chronos can be accessed directly through the object or through the jQuery/Zepto selectors.

var direct = $("#w_methods").chronos(), mind = new Date(), maxd = new Date(); maxd.setDate(maxd.getDate()+7); mind.setDate(mind.getDate()-7); direct.setDateRange({minDate: mind}); // direct access $("#w_methods").chronos("setDateRange", {maxDate: maxd}); // indirect access

With 'daySelected' Callback

Each picker uses callbacks to restrict the range of the sibling picker.

$('.date_picker_callbacks').chronos().on('daySelected', function(event, date, element) { $target = $(event.target) if ($target.attr('id') == "date_picker_callback1") { $("#date_picker_callback2").chronos('setDateRange', {minDate: date}) } else { $("#date_picker_callback1").chronos('setDateRange', {maxDate: date}) } });
