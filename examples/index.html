With Maximum and Minimum Dates Set

Note the code below uses a Date object only to keep with the present date.

var d = new Date(), mind = new Date(), maxd = new Date(), maxStr, minStr; maxd.setDate(d.getDate()+7); mind.setDate(d.getDate()-7); maxStr = maxd.getDate() + "-" + maxd.getMonth() + "-" + maxd.getFullYear(), minStr = mind.getDate() + "-" + mind.getMonth() + "-" + mind.getFullYear(); $('#w_date_range').chronos({ timePicker: true, format: 'j F Y @ h:iA', degbug: true, inputOutputFormat: 'd-m-Y H:i:s', maxDate: { date: maxStr, format: 'd-m-yyyy' }, minDate: { date: minStr, format: 'd-m-yyyy' } });
