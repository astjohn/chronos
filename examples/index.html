Basic

$('#basic').chronos();

With Time Picker

$('#with_time_picker').chronos({useTimePicker: true});

With Maximum and Minimum Dates Set

Note the code below uses a Date object only to keep with the present date.

var mind = new Date(), maxd = new Date(); maxd.setDate(maxd.getDate()+7); mind.setDate(mind.getDate()-7); $('#w_date_range').chronos({ minDate: mind, madDate: maxd });

With Time Picker Callbacks

Each picker uses callbacks to restrict the range of the sibling picker.

$('.time_picker_callbacks').chronos({timePicker: true});
